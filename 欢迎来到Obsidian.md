# 欢迎来到 Obsidian 🎉

恭喜你开始使用 Obsidian 构建个人知识库！

## 📁 仓库结构

```
ObsidianVault/
├── Templates/        # 笔记模板
├── Assets/          # 附件资源
│   ├── Diary/       # 日记附件
│   ├── Articles/    # 文章附件
│   └── Notes/       # 笔记附件
└── .gitignore       # Git忽略配置
```

## 🚀 快速开始

### 1. 双向链接
使用 `[[笔记名称]]` 创建笔记之间的链接

### 2. 标签系统
使用 `#标签名` 为笔记添加分类标签

### 3. 模板使用
- 在 Templates 文件夹创建模板
- 使用 `Ctrl/Cmd + N` 快速创建新笔记

### 4. 知识图谱
按 `Ctrl/Cmd + G` 查看笔记关系网络

## 📝 Markdown 速查

| 功能 | 语法 |
|------|------|
| **加粗** | `**文字**` |
| *斜体* | `*文字*` |
| ==高亮== | `==文字==` |
| ~~删除线~~ | `~~文字~~` |
| `代码` | `` `代码` `` |
| [链接](url) | `[文字](url)` |
