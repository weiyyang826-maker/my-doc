# Telegram 链接测试

## 📱 Telegram 相关链接

### 官方链接
- **Telegram 官网**: https://telegram.org
- **下载页面**: https://telegram.org/apps
- **Web 版**: https://web.telegram.org

### 个人链接格式
```
https://t.me/用户名
tg://resolve?domain=用户名
```

### 群组/频道链接格式
```
https://t.me/群组用户名
tg://join?invite=邀请码
```
