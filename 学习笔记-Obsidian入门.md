# Obsidian 入门指南

> 这是我用 Claude 帮忙创建的第一篇笔记！

## 📌 什么是 Obsidian？

Obsidian 是一款**本地优先**的笔记工具，所有笔记都以 [[Markdown]] 格式存储。

## ✨ 核心功能

### 1. 双向链接
使用 `[[笔记名称]]` 可以创建笔记之间的链接。例如：[[欢迎来到Obsidian]]

### 2. 标签系统
使用 `#标签名` 分类笔记，例如：#学习 #工作 #灵感

### 3. 代码块
```python
def hello_obsidian():
    print("Hello, Obsidian!")
```
